정부 기업인증

정부가 인정하는 우수기업 인증으로 각종 혜택과 지원을 받으세요

기업부설연구소

인증 개요

기업이 자체 R&D 역량을 갖추고 체계적인 연구개발 활동을 수행하는 조직을 보유한 경우 한국산업기술진흥협회(KOITA)에서 인정하는 인증

  • ✓ 연구개발비 세액공제 확대 (최대 50%)
  • ✓ 연구전담요원 병역특례 지정
  • ✓ 정부 R&D 과제 가점
  • ✓ 기술개발 자금 지원 우대

인증 절차

  1. 설립 요건 검토 (연구원 3명 이상, 독립공간)
  2. 연구소 설립 준비 (정관, 운영규정)
  3. 온라인 신청서 작성
  4. 서류 심사 및 현장실사
  5. 인증서 발급 (약 2-3주)
무료 상담 신청

벤처기업 인증

인증 개요

기술성과 성장성이 우수한 중소기업을 벤처기업으로 확인하여 각종 지원혜택을 제공하는 제도

  • ✓ 법인세·소득세 50% 감면 (5년간)
  • ✓ 증권거래세 면제
  • ✓ 스톡옵션 부여 범위 확대
  • ✓ 정부 지원사업 우대
  • ✓ 특허 우선심사

인증 절차

  1. 유형 선택 (기술평가보증, 연구개발, 투자)
  2. 자격 요건 확인
  3. 온라인 신청 (벤처인)
  4. 평가 및 심사
  5. 벤처확인서 발급 (약 2-4주)
무료 상담 신청

이노비즈 (INNOBIZ)

인증 개요

기술혁신형 중소기업(Innovation+Business)으로 기술 우위를 바탕으로 경쟁력을 확보한 기업 인증

  • ✓ 정책자금 금리 우대 (0.3%p)
  • ✓ 보증한도 우대 (최대 70억원)
  • ✓ R&D 지원사업 가점
  • ✓ 조달청 물품구매 적격심사 가점
  • ✓ 병역특례 추천 우대

인증 절차

  1. 자가진단 (온라인 시스템)
  2. 온라인 신청 및 기술평가
  3. 현장평가 (기술·경영혁신)
  4. 종합평가 (650점 이상)
  5. 인증서 발급 (약 4-6주)
무료 상담 신청

메인비즈 (MAINBIZ)

인증 개요

경영혁신형 중소기업(Management Innovation)으로 경영혁신 활동과 성과가 우수한 기업 인증

  • ✓ 신용보증 한도 우대
  • ✓ 정책자금 지원 우대
  • ✓ 수출지원사업 우선 선정
  • ✓ 공공입찰 가점
  • ✓ 광고비 세액공제

인증 절차

  1. 자가진단 실시
  2. 온라인 신청
  3. 경영혁신능력 평가
  4. 현장평가 실시
  5. 인증서 발급 (약 4-6주)
무료 상담 신청

사회적기업

인증 개요

사회적 목적을 추구하면서 영업활동을 수행하는 기업으로, 취약계층 일자리 창출과 사회서비스 제공

  • ✓ 인건비 지원 (최대 3년)
  • ✓ 사업개발비 지원 (연 1억원)
  • ✓ 법인세·소득세 50% 감면
  • ✓ 공공기관 우선구매
  • ✓ 시설비 지원

인증 절차

  1. 인증 요건 확인 (조직형태, 유급근로자)
  2. 인증 신청서 제출
  3. 신청기관 검토 및 현장실사
  4. 인증심사위원회 심의
  5. 고용노동부 인증 (약 3-4개월)
